This is a starter template for [Learn Next.js](https://nextjs.org/learn).

# Create next app

- npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/master/basics/learn-starter"

# Run
- cd nextjs-blog
- npm run dev
- http://localhost:3000/
